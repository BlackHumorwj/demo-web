# 设置模块完善计划

## 目标

为设置模块的11个页面添加完整的CRUD操作功能，包括：搜索筛选、新增、编辑、删除、分页等功能。

## 现状分析

当前设置模块的11个页面均只有基础的列表展示：
- Company.vue - 公司管理
- Employees.vue - 员工管理
- Contacts.vue - 往来单位
- Categories.vue - 收支类别
- Subjects.vue - 科目管理
- Accounts.vue - 账户管理
- Projects.vue - 项目管理
- Departments.vue - 部门管理
- Roles.vue - 角色管理
- Currencies.vue - 币种管理
- Logs.vue - 业务日志（只读）

## 完善方案

### 通用功能模板

每个页面（除Logs外）将添加以下功能：

| 功能 | 描述 |
|------|------|
| 搜索框 | 支持关键字搜索 |
| 筛选器 | 根据状态等字段筛选 |
| 新增按钮 | 弹出表单新增数据 |
| 编辑按钮 | 弹出表单编辑数据 |
| 删除按钮 | 弹出确认框删除数据 |
| 分页 | 支持分页显示 |
| 状态切换 | 快速启用/禁用 |

### 页面优先级

| 优先级 | 页面 | 说明 |
|--------|------|------|
| 高 | Company.vue | 公司信息管理 |
| 高 | Employees.vue | 员工管理 |
| 高 | Contacts.vue | 往来单位管理 |
| 中 | Categories.vue | 收支类别管理 |
| 中 | Subjects.vue | 科目管理 |
| 中 | Accounts.vue | 账户管理 |
| 中 | Projects.vue | 项目管理 |
| 低 | Departments.vue | 部门管理 |
| 低 | Roles.vue | 角色管理 |
| 低 | Currencies.vue | 币种管理 |
| 只读 | Logs.vue | 业务日志（仅展示） |

## 风险评估

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 表单验证复杂 | 用户体验差 | 使用Element Plus表单验证 |
| 数据一致性 | 删除后数据残留 | 弹窗二次确认 |
| 分页状态管理 | 状态混乱 | 使用响应式数据管理 |

## 技术要点

1. 使用 Element Plus 组件：ElForm、ElDialog、ElButton、ElInput 等
2. 使用 Vue 3 Composition API：ref、reactive、computed
3. 弹窗状态管理：通过 ref 控制显隐
4. 表单数据绑定：使用 reactive 对象
5. Mock数据操作：模拟增删改查
